# Security Issues to Find

**Critical (P0/P1)**:
- SQL injection vulnerabilities (parameterized queries missing)
- Cross-site scripting (XSS) - unescaped user input in HTML/JS
- Command injection - user input in system calls
- Path traversal - user-controlled file paths
- Authentication/authorization bypasses
- Session fixation and management issues
- Hardcoded secrets, API keys, passwords in code
- Insecure cryptography (weak algorithms, bad RNG, key management)
- SSRF (Server-Side Request Forgery)
- Deserialization of untrusted data
- Missing input validation on user-controlled data
- CORS misconfigurations allowing unauthorized origins
- CSRF token validation missing

**Important (P2)**:
- Information disclosure (error messages revealing internals)
- Missing security headers (CSP, HSTS, X-Frame-Options)
- Logging sensitive data (passwords, tokens, PII)
- Insecure defaults (debug mode enabled, permissive settings)
- Rate limiting missing on sensitive endpoints
- Weak password policies
- Insufficient logging of security events
